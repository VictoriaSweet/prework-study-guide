Prework Study Guide

HTML

  • The <head> element contains information about the webpage.
  • The <body> element represents the visible content shown to the user
  • An element refers to the entire element from the opening tag to closing tag.
  • A tag refers to only what appears inside the angle brackets (<>).
  • Headers must be placed in chronological order.
  • Keep in mind screen readers accesibility.
  • <p> element represents a paragraph or block text.
  • <ul> element represents unordered lists.
  • <ol> elements represents ordered lists.
  • <li> element represents list items.
  • "src" refers to the location at which the image is stored.
  • "alt" refers to the description of the image displayed.

CSS

  • CSS is used to give 'style' to the webpage.
  • * , aka 'wild card symbol' declares that all of the elements will contain those specific files properties within the brackets.
  • The CSS box model is a common referenc. consisting of 'Margin, Border, Padding and content'
