HTML

  • The <head> element contains information about the webpage.
  • The <body> element represents the visible content shown to the user
  • An element refers to the entire element from the opening tag to closing tag.
  • A tag refers to only what appears inside the angle brackets (<>).
  • Headers must be placed in chronological order.
  • Keep in mind screen readers accesibility.
  • <p> element represents a paragraph or block text.
  • <ul> element represents unordered lists.
  • <ol> elements represents ordered lists.
  • <li> element represents list items.
  • "src" refers to the location at which the image is stored.
  • "alt" refers to the description of the image displayed.

CSS

  • CSS is used to give 'style' to the webpage.
  • The CSS box model is a common reference consisting of Margin, Border, Padding and Content.

Git

  • Git is a tool used to track changes in source code for projects. The following commands are the most commonly used.
  • git status [shows the modified files for your commit]
  • git add [adds a file to the next commit]
  • git checkout [switch to another branch in the working directory]
  • git commit -m [a snapshot of the new commit]

Javascript

  • Javascript makes a webpage interactive.
  • With the use of an array, we're able to store a group of related data in a variable.
  • In a for loop, the first statement makes the starting point for the loop. Declaring a variable and giving it a value.
  • In a for loop, the second statement is the condition.
  • In a for loop, the third statement is how many times the pattern will repeat.
  • Use curly brackets to assign a task you want the function to preform.
